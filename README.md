# To-Do App (Java)

## Overview
I created a simple To-Do application using Java. This project helps users manage their daily tasks. Users can add tasks, view them, mark them as completed, and delete them.

## Features
- Add new tasks
- View all tasks
- Mark tasks as complete
- Delete tasks
- Set reminder time for each task
- Tasks are saved using file handling

## Technologies Used
- Java
- ArrayList
- OOP concepts (class, object, constructor)
- File handling

## How to Run
1. Compile the code:
   javac ToDoApp.java

2. Run the program:
   java ToDoApp
